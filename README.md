## Introducción

Este proyecto está siendo desarrollado por un estudiante de la Universidad de Málaga, actualmente realizando prácticas extracurriculares en la empresa NTT Data.

El repositorio representa una simulación de cómo construir desde cero un pequeño sistema bancario. Para ello, se sigue un enfoque de arquitectura por capas, con responsabilidades bien definidas, aplicando principios fundamentales de ingeniería del software como el bajo acoplamiento y la correcta encapsulación entre sus componentes.

Todo el contenido incluido en este repositorio forma parte de un plan formativo en curso, cuyo objetivo es reforzar los conocimientos prácticos y consolidar conceptos clave. La finalidad es construir una base sólida que permita afrontar con mayor seguridad los retos de proyectos reales en el futuro.
