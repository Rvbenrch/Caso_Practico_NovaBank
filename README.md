# NovaBank Digital Services

## Descripción

Aplicación de consola en Java que simula un sistema básico de gestión bancaria.

En esta primera versión el sistema funciona completamente en memoria, sin base de datos ni autenticación.

## Funcionalidades implementadas

### Gestión de clientes
- Creación de clientes
- Validación de DNI único
- Almacenamiento en memoria mediante ArrayList
- Generación automática de ID
